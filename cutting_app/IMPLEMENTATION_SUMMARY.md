# Реализация ИС Раскрой — сводка

## Общая информация

- **Проект**: ИС Раскрой — информационная система оптимального раскроя листовых материалов
- **Тип**: Десктопное приложение (Python + PyQt6)
- **База данных**: SQLite (SQLAlchemy 2.0)

---

## Реализованные модули

| Модуль | Файлов | Описание |
|--------|---------|----------|
| db/ | 4 | Модели данных (SQLAlchemy) |
| core/ | 6 | Бизнес-логика (сервисы) |
| algorithms/ | 5 | Алгоритмы оптимизации |
| ui/ | 15 | Графический интерфейс |
| utils/ | 3 | Утилиты |

## Реализованные функции

### 1. Справочники
- [x] Материалы (название, плотность, текстура)

- [x] Учёт остатков листов на складе

### 3. Заказы
- [x] Создание заказов с многопозиционными спецификациями
- [x] Импорт спецификаций из CSV
- [x] Импорт спецификаций из Excel (xlsx)

- [x] Создание задания на раскрой

- [x] Параметры алгоритмов (поколения, мутации, температура и т.д.)

- [x] Экспорт в PDF (карты раскроя)

### 8. Тестирование

- [x] Unit-тесты для сервисов

## Следующие шаги

1. Запустить pytest с покрытием:
   ```bash
   pytest --cov=. --cov-report=term-missing
   ```

2. Запустить нагрузочные бенчмарки:
   ```bash
   python scripts/acceptance_benchmark.py --pieces 200
   python scripts/acceptance_benchmark.py --pieces 500
   ```

3. При необходимости — оптимизировать алгоритмы

4. Собрать дистрибутив:
   ```bash
   pyinstaller build.spec
   ```

## Команды запуска

```bash
# Установка зависимостей
pip install -r requirements.txt

# Запуск приложения
python main.py

# Заполнение тестовыми данными
python seed_data.py --reset

# Тесты
pytest -v

# Покрытие тестами
pytest --cov=. --cov-report=term-missing

# Бенчмарк
python scripts/acceptance_benchmark.py --pieces 200
```
