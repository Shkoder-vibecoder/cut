# ИС Раскрой - Информационная система оптимального раскроя листовых материалов

Десктопное приложение для автоматизации процесса формирования оптимальных карт раскроя листовых материалов (2D Bin Packing / Cutting Stock Problem).

## Возможности

- Ведение справочников материалов и форматов листов
- Учёт складских остатков с автоматическим списанием
- Создание и управление заказами с многопозиционными спецификациями
- Импорт спецификаций из CSV и Excel
- Оптимизация раскроя тремя алгоритмами: жадная эвристика (FFD), генетический алгоритм, имитация отжига
- Визуализация карт раскроя с масштабированием
- Экспорт результатов в PDF и PNG
- Генерация этикеток с QR-кодами

## Требования

- Python 3.11+
- PyQt6
- SQLAlchemy 2.0+
- numpy 1.26+
- openpyxl 3.1+
- reportlab 4.0+
- Pillow 10.0+
- qrcode 7.4+
- python-barcode 0.15+

## Установка

1. Клонируйте репозиторий
2. Установите зависимости:
```
pip install -r requirements.txt
```

## Запуск

```
python main.py
```

## Заполнение тестовыми данными

```bash
python seed_data.py --reset
```

Скрипт создаёт тестовые материалы, склад, заказы и изображение-превью листа в `assets/seed_sheet_preview.png`.

## Структура проекта

```
cutting_app/
├── main.py                 # Точка входа
├── config.py               # Конфигурация
├── requirements.txt        # Зависимости
├── db/                     # Модели данных (SQLAlchemy)
├── core/                   # Бизнес-логика (сервисы)
├── algorithms/            # Алгоритмы оптимизации
├── ui/                    # Графический интерфейс (PyQt6)
├── utils/                 # Утилиты (импорт CSV/Excel)
└── tests/                 # Unit-тесты
```

## Основной сценарий работы (3 шага)

1. **Склад** - выбрать листы материала
2. **Заказы** - загрузить список деталей (или импортировать CSV/Excel)
3. **Задание** - нажать «Рассчитать раскрой»
